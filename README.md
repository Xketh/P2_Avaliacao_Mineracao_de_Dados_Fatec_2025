# P2_Avaliacao_Mineracao_de_Dados_Fatec_2025
Projeto individual da P2 de Mineração de Dados do curso DSM – 6º semestre (Fatec, 2025), orientado pelo professor Lucas Bonilha Henrique. Utilizando o Groceries Dataset (Kaggle) para analisar compras da empresa Market+, aplicando regras de associação, clusterização e classificação, afim de apoiar decisões estratégicas
